# Controlling 5-12V devices with Arduino

How to trigger/control 12V devices (motors, LEDs) with Arduino

*NOTE: Arduino **VIN pin** stands for "Voltage-IN", supporting up to 12V. However, this is not an output.* 

### 5-12V Actuators:

- motors

- LED Strips

- fans / pumps
